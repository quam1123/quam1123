    2.用自定义函数传递函数名 瞎起的名字
    大概需求：五项求和（自然数/自然数的立方）

    def cube(k):
    return pow(k,3)
def identity(k):
    return k
def sum_naturals(n,term):
    sum,k = 0 , 1
    while k <= n:
        sum , k = sum + k , k + 1
    return sum

    通过了“term”传递了函数，使得我们不需要分别写两个自定义函数分别进行自然数/立方的求和
    函数也可以作为return 后的值被返回
